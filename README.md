# DSA Mastery: Python Solutions
This repository contains optimized solutions to core algorithmic problems, focusing on time and space efficiency.

## 1. Two Sum
*   **Approach**: Hash Map (One-pass)
*   **Time Complexity**: $O(n)$
*   **Space Complexity**: $O(n)$
*   **Why**: Optimized from $O(n^2)$ by storing previously seen values in a dictionary.

## 2. LRU Cache
*   **Approach**: OrderedDict (Hash Map + Doubly Linked List logic)
*   **Time Complexity**: $O(1)$ for get/put
*   **Why**: Efficient memory management by removing the "Least Recently Used" item.

## 3. Level Order Traversal
*   **Approach**: Breadth-First Search (BFS) using a Queue
*   **Time Complexity**: $O(n)$
*   **Why**: Essential for processing hierarchical data structures like AI training trees.
